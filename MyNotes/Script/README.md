## 📋 О проекте

**Git Repository Sync Tool** — это мощный кроссплатформенный инструмент для автоматической синхронизации содержимого между Git-репозиториями. Скрипты написаны на **Bash** (для Linux/macOS/WSL) и **PowerShell** (для Windows), что обеспечивает полную совместимость со всеми основными операционными системами.

Инструмент предназначен для автоматизации процесса копирования содержимого одного репозитория в другой с сохранением структуры, исключением служебных папок (`.git`) и опциональной синхронизацией с удаленными репозиториями.

### 🎯 Ключевые возможности

| Функция | Описание |
|---------|----------|
| 🔄 **Git pull** | Автоматическая синхронизация с удаленным репозиторием перед копированием |
| 📁 **Интеллектуальное копирование** | Копирование всего содержимого, кроме папки `.git` и служебных файлов |
| 🚀 **Опциональный push** | Автоматический git push в целевом репозитории после копирования |
| 🔒 **Приватные репозитории** | Возможность пропуска или специальной обработки приватных репозиториев |
| 🖥️ **Визуальный контроль** | Запуск в отдельном окне терминала для наблюдения за процессом |
| 🛡️ **Безопасность** | Проверка незакоммиченных изменений и подтверждение операций |
| 📊 **Цветной вывод** | Наглядное отображение статуса выполнения (успех/ошибка/предупреждение) |
| ✅ **Валидация путей** | Проверка существования и корректности всех указанных путей |
| 🔍 **Git-валидация** | Проверка, что указанные директории являются Git-репозиториями |

### Установка

<details>
<summary><b>🐧 Linux / macOS / WSL</b></summary>

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/git-repo-sync.git
cd git-repo-sync

# Сделать скрипт исполняемым
chmod +x sync-repo.sh

# Опционально: добавить в PATH для глобального доступа
sudo cp sync-repo.sh /usr/local/bin/
